# Agent Instructions

This repository contains instruction sets and guidelines for AI coding assistants working on the `co-own` repository.

## GitHub Workflow Directive
- **Always push all changes to GitHub `main` branch.** After completing any coding, modifying, or debugging tasks, verify the changes and immediately stage, commit, and push them to the `main` branch of the remote repository (`origin main`).

## Asset Management Directive
- **NEVER use local media files.** All images, videos, or other media assets must be hosted on the public internet (such as raw GitHub URLs from public repositories) rather than stored locally in the workspace or committed to this repository.
- **Do not commit local image assets.** All files should be visible on the public internet and not rely on files stored on a local computer.
